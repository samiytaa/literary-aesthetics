# literary-aesthetics

这是一个围绕“文字风味轮”主题制作的静态网页项目。

## 项目来源

本仓库基于原项目
[ashlynlyq513-cyber/literary-aesthetics](https://github.com/ashlynlyq513-cyber/literary-aesthetics)
进行修改，并由
[samiytaa](https://github.com/samiytaa)
在此 fork 中继续维护与发布。

## 本仓库修改内容

- 在原项目基础上继续进行页面内容与展示效果调整
- 保留原项目来源信息，并作为独立 fork 版本发布
- 使用 GitHub Actions 自动部署到 GitHub Pages

## 部署方式

本项目通过 GitHub Actions 自动部署到 GitHub Pages。
